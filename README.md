# 📋 Task Manager

A simple and responsive Task Management Web Application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://mani887035.github.io/Task-Manager/

## 📌 Features

- Add new tasks
- Mark tasks as completed
- Undo completed tasks
- Delete tasks
- Responsive design for desktop and mobile
- Local Storage support (tasks remain after page refresh)
- Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 🎯 How It Works

1. Enter a task in the input field.
2. Click **Add Task** to create a new task.
3. Click **Complete** to mark a task as completed.
4. Click **Undo** to revert a completed task.
5. Click **Delete** to remove a task.
6. Tasks are automatically saved in the browser's Local Storage.
